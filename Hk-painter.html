HK-Painterwebsite

Coursework for my diploma

I made this website as part of my coursework for my diploma in software development. The site was made using HTML5, CSS3, CSS grid, Bootstrap and jsdeliver. The site was designed with a mobile first methodology.

You can visit the site here.

There are three pages contained within the site. A homepage, a portfolio and a contact page. The home page below details what the painter does, some of the work she has done (presented within a carousel ) and where you can find her (presented using an iframe and Google maps). All pages use the nav and footer elements.

The portfolio page details some of the jobs Helen has done. I used cards with images to present these. When the user hovers over the card, it scales up in size to help them focus on that particular card.

The contact page consists of a simple form where the user can fill out their name, phone number, email and a description of the project they need completing. The name and phone number inputs are mandatory.

Each page is designed to be easy to navigate, using the nav and buttons to clearly show and direct the user to the next part of the page. This is supported by by hover effects to give visual feedback to the user, so they know what element they are interacting with.
